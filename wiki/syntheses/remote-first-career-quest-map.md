# Remote-First Career Quest Map

## Summary

This map turns the goal of becoming a remote-first [[concepts/remote-first-frontend-fullstack-engineer|frontend-based fullstack engineer]] into a set of playful, artifact-producing what-if quests.

The map has three axes:

1. **Frontend Taste** — visible product craft.
2. **Fullstack Ownership** — end-to-end feature credibility.
3. **Remote Trust** — async collaboration confidence.

## Capability axes

| Axis | What it proves | Useful artifacts |
| --- | --- | --- |
| Frontend Taste | The user can make software feel clear, polished, and usable. | interactive demo, component notes, accessibility pass, UI before/after |
| Fullstack Ownership | The user can ship a coherent product slice across UI/API/data/deploy. | architecture diagram, schema note, API docs, tests, deployed feature |
| Remote Trust | The user is easy to work with asynchronously. | async proposal, tradeoff log, decision note, PR summary, demo script |

## Initial quest seeds

### 1. Remote Trust

> What if I could prove remote trust before the first interview?

Loot:

- async feature proposal
- tradeoff note
- PR-style summary
- demo script
- “How I work async” portfolio section

### 2. Frontend Taste

> What if one small interface could make a hiring manager feel my frontend taste in ten seconds?

Loot:

- polished UI demo
- state coverage: loading, empty, error, success
- component breakdown
- short design decision note

### 3. Fullstack Ownership

> What if a frontend engineer could prove they can own a feature end-to-end before the interview?

Loot:

- small shipped product slice
- API/data model note
- test plan
- deploy link
- architecture diagram

### 4. Interview Boss Fight

> What if I answered interview questions from artifacts instead of memorized scripts?

Loot:

- interview story bank
- STAR answers from real quest outputs
- tradeoff stories
- failure and recovery stories

### 5. Rejection Alchemy

> What if rejection emails became XP telemetry?

Loot:

- rejection postmortem template
- next experiment card
- adjusted targeting signal
- confidence-preserving feedback loop
